# PE FILE READER

필요 모듈 : pefile, PyQt5

- pefile

  pe 파일 포맷을 가져오기 위한 Module

  - 설치 명령어

    ```
    $sudo pip3 install pefile
    ```

    ​

- PyQt5

  GUI 프로그래밍을 위한 Module

  - 설치 명령어

    ```
    $sudo pip3 install PyQt5
    ```

    ​

- 통합 모듈 설치 명령어

  ```
  sudo pip3 install pefile
  sudo pip3 install PyQt5
  ```

  ​

- 실행 명령어

  ```
  python3 PE_Reader.py
  ```

  ​
